Connect to Google Sheets

One-time setup — your data saves permanently to a Google Sheet you own, accessible from any device.

1
Create a blank Google Sheet
Go to sheets.google.com, click + to create a blank spreadsheet, name it Watch Business.
💡 Make sure you're signed into your Google account.
2
Add the backend script
In your sheet click Extensions → Apps Script. Delete everything in the editor, paste the entire contents of watchbiz_backend.gs, and save.
3
Run setupSheets once
Select setupSheets from the function dropdown → click ▶ Run → accept permissions (click Advanced → Go to project → Allow).
💡 You'll see "✅ All sheets created!"
4
Deploy as Web App
Deploy → New deployment → Type: Web app → Execute as: Me → Access: Anyone → Deploy → copy the URL.
5
Paste your Web App URL
