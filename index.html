Connect to Google Sheets

One-time setup — your data will be saved permanently in a Google Sheet you own, accessible from any device.

1
Create a blank Google Sheet
Go to sheets.google.com, click the big + to create a blank spreadsheet, and name it Watch Business (or anything you like).

The script will automatically create all the tabs inside it in Step 3.
💡 Make sure you're signed in to your Google account first.
2
Open the Apps Script editor
In your copied sheet, click the menu: Extensions → Apps Script. A new tab opens with code already in it.
💡 You'll see a file called Code.gs with the backend code. Don't change anything.
3
Run the setup function once
In the Apps Script editor, select setupSheets from the function dropdown (top center), then click the ▶ Run button. Accept the permissions popup — this lets the script read/write your sheet.

You'll see a popup: "✅ All sheets created!"
4
Deploy as a Web App
Still in Apps Script, click Deploy → New deployment.
— Type: Web app
— Execute as: Me
— Who has access: Anyone
Click Deploy, accept permissions again, then copy the Web App URL.
💡 The URL looks like: https://script.google.com/macros/s/ABC.../exec
5
Paste your Web App URL here
Paste the URL you just copied and click Connect.
